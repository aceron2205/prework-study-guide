HTML

  • Enter your HTML notes here
  • The head element contains information about the webpage.
  • The body element represents the visible content shown to the user.
  • HTML developers are responsible of providing structure and provide a positive user experience.
  • Children elements are those that are nested inside other elements.
  • The elements that hold the children are call parent elements.

CSS

  • CSS language provides style to the content.
  • There are three ways to style a webpage with CSS: inline css, internal css style sheet and external css.
  • When external css you have to link it with the tags "link rel:"stylesheet" href="stylesheet.css".
  • The internal style sheet and external css has to be added between the parent element "head"
  • Selectors are used to "find" HTML elements you want to style.
  • A margin indicates how much space we want around the outside of an element.
  • A padding indicates how much space we want around the content inside an element.

Git

  • Git allow us to work from our local environment without making changes to the main branch.
  • We do that by following the next steps:
      • Create a new branch.
      Add commits
      Open a pull request (PR), it good practice to have a team mate review your changes.
      Discuss and review
      Merge and deploy
  • Git commands are always preceded by a Git statement.
  • Checkout command allow us to move to a working branch.
  • It is good practice to add a prefix to a new branch in order to differentiate it from the main branch.

JavaScript

  • You can group data in an array
  • The array indece follow a predictable pattern starting with zero and growing by 1 (no decimals).
  • JavaScript provides logic and allows users to interact with the website.
  • Funcions are the building blocks of JavaScript.
  • In function, we define a block of code that performs a specific task and give it a unique name.
  • In order to execute a function we must call it.
  • JavaScript's control flow runs from top to bottom.
  • You can interrupt control flow with for functions
